TravelController.java: Этот класс будет обрабатывать HTTP-запросы.
Вам нужно будет реализовать методы для обработки запросов GET, POST, PUT и DELETE.

Employee.java, Travel.java, Expense.java: Это будут сущности JPA, представляющие таблицы базы данных.
Вам нужно будет определить поля и методы доступа для каждого из них.

TravelService.java: Этот класс будет содержать бизнес-логику приложения.
Вам нужно будет реализовать методы для расчета командировочных, основываясь на нормах по ссылке.

EmployeeRepository.java, TravelRepository.java, ExpenseRepository.java: Это будут репозитории Spring Data JPA для взаимодействия с базой данных.
Вам нужно будет определить методы для выполнения операций CRUD.

TravelAppApplication.java: Это будет основной класс приложения.
Вам нужно будет реализовать метод main, который запускает приложение.

application.properties: Этот файл будет содержать настройки приложения, такие как параметры подключения к базе данных.

TravelAppApplicationTests.java: Этот класс будет содержать тесты для вашего приложения.
Вам нужно будет реализовать тесты для проверки корректности работы вашего приложения.

pom.xml: Этот файл будет содержать зависимости и настройки сборки проекта.
Вам нужно будет добавить необходимые зависимости и настроить сборку проекта.
